qrComparator

Project Description

This project uses a modified larsung.de's jQuery.qrcode-0.8.0.js as the QR Code generator in conjunction with LazarSoft's jsqrcode to decode QR codes. The purpose of this project is to display visual differences between the original QR code and a modified QR code, and will attempt to visually depict a variety of attacks on QR codes. For layout purposes, this page is formatted with Twitter BootStrap and aimed to be mobile-responsive.
